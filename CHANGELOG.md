# Changelog

## v5.0-free

- Switched the project direction to a free public OpenFX plugin.
- Removed earlier commercial distribution experiments, service-side tooling, beta operator runbooks, packaged binary archives, and runtime restriction paths.
- Kept the core Resolve OpenFX effect, 54 film presets, DCTL/LUT source resources, build scripts, install scripts, and focused tests.
- Rewrote public documentation for GitHub release and open development.
