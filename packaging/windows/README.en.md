# DuXun Film Simulation v5.0 Free Windows Package

This one-click package is for Windows users who do not want to build the plugin from source.

## Install

1. Close DaVinci Resolve.
2. Extract the whole zip. Do not drag out only one file.
3. Double-click `Install.bat`.
4. If Windows asks for administrator permission, choose Yes.
5. Restart DaVinci Resolve after installation.
6. Find the effect under:

```text
OpenFX > DuXun > DuXun Film Simulation
```

## Uninstall

Double-click `Uninstall.bat` and approve administrator permission when prompted.

## Troubleshooting

- If double-clicking does nothing, right-click `Install.bat` and choose "Run as administrator".
- If the installer says DaVinci Resolve is running, close Resolve and run the installer again.
- If Resolve does not show the plugin, confirm this folder exists:

```text
C:\Program Files\Common Files\OFX\Plugins\DuXunFilmSim.ofx.bundle
```
