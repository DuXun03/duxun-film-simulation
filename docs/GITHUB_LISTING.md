## Public Summary

DuXun Film Simulation is a free OpenFX plugin for DaVinci Resolve. It combines 54 built-in film presets with practical controls for curve shaping, density, print response, grain, halation, bloom, film damage, film breath, gate weave, overscan, and vignette. The plugin runs locally, requires no account or network service, and keeps the source code, build scripts, and install scripts in the repository.

## Key Selling Points

- Free local Resolve workflow with no account gate
- 54 built-in film presets across Agfa, CineStill, Fuji, Ilford, and Kodak families
- One OpenFX effect instead of scattered DCTL installation
- Chinese UI labels for fast color-grading sessions
- Film texture controls: grain, halation, bloom, damage, breath, weave, overscan, vignette
- Source-first repository for learning, auditing, and extending the effect

## First Release Notes

This free public release removes the earlier paid distribution experiments and focuses the repository on the OpenFX plugin itself. The source tree now keeps the core C++ effect, build/install scripts, presets, DCTL resources, tests, and concise public documentation.
